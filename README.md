# Luke Warner Career Website

Personal portfolio website for customer success, implementation, and customer-facing startup roles.

## Local Preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Netlify

This is a static site. Netlify should use:

- Build command: none
- Publish directory: `.`
